#  Multi-Horizon Sea Level Forecasting using ML & LSTM

##  Overview
This project predicts Mean Sea Level (MSL) at 1, 6, 12, and 24 hours ahead using time-series data and meteorological variables.

##  Techniques Used
- Time-series preprocessing
- Lag & rolling feature engineering
- Machine Learning (RF, XGBoost, SVR, KNN)
- Deep Learning (LSTM)
- Monte Carlo Dropout for uncertainty

##  Models
- Random Forest
- XGBoost
- SVR
- KNN
- LSTM

##  Results
- Evaluated using RMSE, MAE, R²
- Compared ML vs LSTM performance
- Generated confidence intervals

##  How to Run
1. Open notebook in Google Colab
2. Run all cells
3. View predictions and plots

##  Dataset
Custom dataset with sea level and meteorological variables
